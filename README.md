## Usage

- Install Poetry

    ```bash
    pipx install poetry
    ```

- Clone Repo

    ```bash
    git clone https://github.com/wuihee/machine-learning-algorithms.git
    cd machine-learning-algorithms
    ```

- Install Dependencies

    ```bash
    poetry install
    ```
